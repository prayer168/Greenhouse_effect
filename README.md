# 溫室效應探究與實作

這是一份國小自然互動數位教材，參考「獨居蜂的秘密生活」的單元式架構，改為溫室效應主題。

## 教材入口

開啟 `index.html` 即可使用。教材為純靜態網頁，包含：

- 學習目標與單元導覽
- 自然溫室效應與增強溫室效應比較
- 可操作的溫室效應概念模型
- 校園減碳方案設計任務
- 科學證據與迷思判斷
- 8 題闖關測驗
- 各單元 SVG 或 JavaScript 動畫提示，包含能量流程、熱量回射、行動進度、證據天平與闖關徽章

## 建議教學流程

1. 先請學生回答核心問題：「溫室效應是壞事嗎？」
2. 使用概念比較單元建立自然與增強溫室效應的差異。
3. 在模擬實驗中分組調整二氧化碳、甲烷、反射率與節能比例，記錄預測和觀察結果。
4. 讓小組設計 4 個以內的校園減碳行動，說明可行性。
5. 用證據判斷和闖關測驗進行形成性評量。

## 模型限制

模擬中的保留熱量與溫度變化是教學用概念模型，目的是呈現變因方向與比較趨勢，不是正式氣候預報或真實氣候模式。
